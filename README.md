# Computer Architecture Exercises

This repository contains ARM assembly exercises and lab work for computer architecture practice.

## Structure

- `ARM/` - Example ARM projects and instruction-based exercises.
- `Labs/` - Lab assignments with ARM assembly solutions and problem descriptions.

## Contents

- Assembly source files for ARM processor exercises.
- Problem statements and task descriptions for lab work.
- Organized folders for each exercise and lab.

The repository is focused on learning ARM assembly and exploring computer architecture concepts through practical examples.
